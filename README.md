In today’s digital era, video-based content has become an essential part of everyday life, transforming how people learn, work, and entertain themselves. Students depend on recorded lectures and online learning platforms, professionals engage in virtual meetings and training sessions, and individuals consume multimedia content for extended periods. Although this shift has greatly improved accessibility and convenience, it has also introduced a significant challenge—maintaining consistent user attention throughout the duration of the content. Human attention span is naturally limited, and during long viewing sessions, users often lose focus due to fatigue, distractions, or drowsiness. Consequently, important information may be missed while the video continues to play, reducing the overall effectiveness of the content.

To overcome this limitation, the proposed Attention-Aware Video Player presents an intelligent and adaptive solution to enhance user engagement. Unlike traditional video players that function passively, this system actively monitors the user in real time through a standard webcam. By analyzing facial behavior and visual cues, it determines whether the viewer is attentive or distracted. When the system detects signs of inattention—such as closed eyes, frequent blinking, or the user looking away—it automatically pauses the video. Playback resumes only when the system confirms that the user has regained focus, ensuring that no critical content is missed.

The system is developed using core computer vision techniques that allow efficient real-time analysis without requiring costly hardware or continuous internet access. Facial landmark detection is employed to identify key points on the user’s face, forming the basis for further evaluation. The Eye Aspect Ratio (EAR) is used to detect drowsiness by analyzing eye closure patterns, while head pose estimation determines the direction of the user’s gaze. By integrating these techniques, the system accurately assesses user attention in a non-intrusive and user-friendly manner.

Overall, this project demonstrates the practical application of computer vision in enhancing user experience, making video consumption more interactive, responsive, and aligned with human behavior.
